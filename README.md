# React Expense Tracker App

- Add new expense
- Expense month chart
- Filter expenses by year
- List expenses.

![alt text](https://github.com/serhatkaya/serhatkaya/raw/main/app-images/expense-tracker.png)

# How to run

- You must have at least one of these package managers [npm, yarn]

# npm

- npm install
- npm start

# yarn

- yarn install
- yarn start
